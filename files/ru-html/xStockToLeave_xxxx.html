Определяет количество материала, оставляемое в радиальном направлении.
Используется для оставления припуска на внутреннем (ВД) или наружном (НД) диаметре детали.

Припуск по X 1мм

Припуск по X 1мм

Припуск по X 0.0

Припуск по X и Z -1мм

Для поверхностей, которые не являются строго горизонтальными, программа интерполирует между значением Осевого припуска (стенка) и значениями Радиального припуска. Припуск, оставляемый в радиальном направлении на этих поверхностях, может отличаться от указанного значения в зависимости от уклона поверхности и значения осевого припуска.

Изменение значения Припуска по X автоматически устанавливает Припуск по Z на ту же величину. Вы можете вручную ввести другое значение Осевого припуска.

Если вы планируете черновую и чистовую обработку разными инструментами, установите для черновой операции величину припуска для следующей чистовой операции.

Отрицательный припуск Вы можете использовать отрицательное значение. Это приведет к тому, что инструмент будет обрабатывать материал за пределами кромки вашей модели.

Примечание: Максимальный отрицательный припуск должен быть меньше радиуса вершины пластины.
При радиусе вершины пластины 0,032 мм наибольший отрицательный припуск будет -0,032 мм.
Вы не можете компенсировать за теоретическую вершину инструмента.
