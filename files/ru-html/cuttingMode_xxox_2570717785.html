Используется для станков со встроенными таблицами Качества для установки соответствующих подач резания обрабатываемого материала.

* Сквозной - Авто
* Сквозной - высокое качество
* Сквозной - среднее качество
* Сквозной - низкое качество
* Гравировка
* Выжигание

Примечание: Гравировка наносит marks на поверхность материала без сквозного реза.
Выжигание используется для удаления пленки с некоторых материалов.
Некоторые опции недоступны для всех типов станков.
Пример: Нельзя гравировать плазменным резаком.
