Наклоняет инструмент вбок относительно плоскости, перпендикулярной направлению движения.

1 - Направление движения
2 - Направление бокового наклона

Угол наклона используется для небольшого наклона оси инструмента в сторону детали, чтобы резать большей площадью инструмента. В примере ниже это позволяет использовать большие значения шага погружения, сокращая время обработки.

Угол опережения / запаздывания = 0
Угол наклона = 0
Шаг погружения = 0.5 мм

Угол опережения / запаздывания = 0
Угол наклона = -88
Шаг погружения = 1 мм
