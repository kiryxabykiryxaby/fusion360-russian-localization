Позволяет включить утолщение, примененное к структурам поддержки, в файл 3MF для имитации процесса.

Это относится только к структурам поддержки, которые созданы с выбранной опцией "Утолщенные верхние структуры". Снятие флажка означает, что для имитации процесса используется предварительно утолщенная замещающая сетка поддержки.
